
# 🏠 House Price Prediction System

## 📌 Project Description

The **House Price Prediction System** is a Machine Learning project developed to estimate the price of a house based on different housing features such as area, number of bedrooms, bathrooms, and other property attributes. The main objective of this project is to build a predictive model that can estimate house prices accurately using historical housing data.

The project starts with **Exploratory Data Analysis (EDA)** to understand the dataset, identify patterns, and analyze relationships between different features that influence house prices. Visualization techniques are used to study data distribution, correlations between variables, and detect important factors affecting the house price.

After analyzing the dataset, **data preprocessing** is performed to clean and prepare the data for machine learning. This step may include handling missing values, selecting relevant features, and converting categorical data into numerical format if required.

The main algorithm used in this project is **Linear Regression**, which is a supervised learning algorithm used for predicting continuous numerical values. Linear Regression works by finding the best linear relationship between independent variables (input features) and the dependent variable (house price). The model learns patterns from the dataset and uses them to estimate prices for new inputs.

Once the model is trained, it is evaluated using regression performance metrics such as **Root Mean Squared Error (RMSE)** and **R² Score** to measure the prediction accuracy.

The trained model is then integrated into a **Streamlit web application**, allowing users to enter housing details and receive predicted house prices instantly. The web interface makes the system interactive and easy to use.

This project demonstrates the complete **Machine Learning pipeline**, including:

* Data preprocessing
* Exploratory Data Analysis (EDA)
* Feature selection
* Model training using Linear Regression
* Model evaluation
* Deployment using Streamlit

The project highlights how **Machine Learning can be applied in the real estate industry to estimate property prices and support data-driven decision making**.
